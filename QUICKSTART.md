### 修改配色方案
編輯 `styles.css` 第1-10行:
```css
:root {
    --primary: #00FFE0;      /* 改成你的主色 */
    --river-blue: #8BCAFE;   /* 改成你的次要色 */
    --leaf-green: #96D1C3;   /* 改成你的輔助色 */
}
```

### 開關特定效果


## 🐛 疑難排解

### Q: 看不到動畫效果？
A: 確認已引入 `interactive-effects.js`，檢查瀏覽器控制台錯誤訊息

### Q: 粒子沒有顯示？
A: 檢查CSS是否正確載入，確認 `.particles-container` 樣式存在

### Q: 地圖點擊沒反應？
A: 確認 `script.js` 正確載入，檢查地圖標記的 `data-country` 屬性

### Q: 計數器不會動？
A: 確認元素有 `data-target` 屬性，且 `counter` class 存在

### Q: 手機上效果簡化？
A: 這是正常的響應式優化，為了保證效能
